# HR-Analytics-Dashboard

## Overview
This project is an interactive Power BI dashboard developed to analyze workforce data and HR performance metrics.

## Features
- Employee attrition analysis
- Department-wise employee distribution
- Salary analysis
- Employee performance tracking
- KPI cards
- Interactive visualizations and filters

## Tools Used
- Power BI
- Microsoft Excel

## Insights
The dashboard helps identify:
- Attrition trends
- Workforce distribution
- Department performance
- Employee-related insights
